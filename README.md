# AI-Powered Fake News Detection Using Text Classification

## Project Overview

Fake news has become a major problem with the growth of online information. This project uses Artificial Intelligence and Machine Learning techniques to automatically classify news articles as **Fake or Real** using Natural Language Processing (NLP).

The system analyzes news text, extracts important features, trains machine learning models, and predicts whether a given news article is fake or genuine.

---

## Objective

The main objective of this project is to build a machine learning pipeline that can:

- Clean and preprocess news text
- Convert text into numerical features
- Train different machine learning algorithms
- Evaluate model performance
- Predict fake or real news

## Dataset

Dataset used:

Fake News Detection Dataset

Dataset contains news articles with text information and labels.

Features:

- Title
- Text
- Subject
- Date
- Label

Label:

- 0 → Fake News
- 1 → Real News
